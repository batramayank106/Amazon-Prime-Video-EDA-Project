# Contributing to Amazon Prime Video EDA

Thank you for your interest in contributing to this project.

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

## Guidelines

- Follow PEP 8 style for Python code
- Ensure all visualizations render correctly
- Update the README if adding new analysis sections
- Add descriptive markdown before and after each visualization
- Keep the notebook organized with the existing Table of Contents structure

## Reporting Issues

- Use the GitHub Issues tab to report bugs or suggest improvements
- Include the notebook cell number where the issue occurs
- Provide steps to reproduce the problem

## Code of Conduct

Be respectful and constructive in all interactions.
