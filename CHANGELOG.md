# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2025-11-01

### Added

- Initial release of Amazon Prime Video EDA project
- Exploratory Data Analysis notebook with 16+ EDA sections
- Data cleaning: duplicate removal, missing value handling, text standardization, data validation
- Statistical analysis: Correlation, Chi-Square, ANOVA, t-test, Mann-Whitney U, Shapiro-Wilk
- Feature engineering: decade, content era, genre/country parsing, rating categories
- Interactive Plotly visualizations with dark theme
- Matplotlib and Seaborn static charts
- Saved chart images for README integration
- Executive Summary, Business Objectives, and Final Report
- Data quality assessment: duplicates, missing values, memory optimization
- Complete data dictionary for titles.csv and credits.csv
- Project support files: requirements.txt, environment.yml, LICENSE, .gitignore
